# samples/data — 원본 PDF는 저장소에서 제외됨

이 폴더의 **원본 PDF는 용량(약 44MB) 때문에 git에 커밋하지 않습니다**
(`.gitignore`: `samples/data/*.pdf`). 결과를 **보는 데는 PDF가 필요 없습니다** —
정형 CSV·청크·Chroma 인덱스는 `samples/outputs/`에 커밋되어 있어, 부트스트랩만 하면
신호등 대시보드·RAG 검색이 **키 없이** 바로 동작합니다.

## 원본을 다시 받아 0단계부터 재현하려면

대상 문서는 환경부·한국환경산업기술원(KEITI)이 매년 공개하는
**「친환경 생활·소비 국민 인지도 조사 결과보고서」** 입니다(공식 홈페이지·공공데이터 공개 자료).
현재 구축 대상은 다음 3개년입니다.

- 2023년 인지도조사 결과보고서
- 2024년 친환경 생활 소비 국민 인지도 조사 결과보고서
- 2025년 친환경생활·소비 국민 인지도 조사 결과보고서

내려받은 PDF를 작업 폴더 `data/`(또는 이 `samples/data/`)에 두고
`README.md`의 **빠른 시작 B) 처음부터 / 내 데이터로** 절차를 따르면 추출→표준화→인덱싱을
직접 재현할 수 있습니다.
